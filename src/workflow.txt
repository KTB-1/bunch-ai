# NewsDB 채우기
1. main.py 실행 (저는 config 설정해서 가져오는 개수는 5개로 했어요)

# usernewsview 채우기
2. embed_news.py 실행

3. DBeaver에서 userID 임의로 변경 (embed_news에서는 1번 유저가 모든 뉴스 본 것으로 테이블 채웠음)
(ex) userid 1로 되어있는거 적당히 다른 userid로 변경
viewid  userid  newsid  viewdate
1       1       1       2024-08-29 01:15:41
2       1       2       ...
3       1       3
4       2       4
5       2       5

# chromaDB 서버 실행
4. 터미널 새로 열어서 """ chroma run --path "폴더 경로" """

# flask 서버 실행
5. 터미널 새로 열어서 app.py 실행

# post 요청 보내기
6. 터미널 새로 열어서 명령어 입력
curl -X POST -H "Content-Type: application/json" -d '{"userid": 1, "cnt": 3}' http://localhost:5000/get_recommendations

userid는 유저 번호, cnt는 추천할 개수 limit
